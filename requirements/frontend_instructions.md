# Project overview
Use this guide to build a web app where users can input Tamil text or upload a file containing Tamil text to generate the Braille equivalent using a Tamil-to-Braille conversion mapping file


# Feature requirements
- We will use Next.js, Shadcn, Lucide, Supabase, Clerk
- Create a form where users can type Tamil text or upload a .txt or .pdf file
- When the user clicks the convert button, the app should convert Tamil text to Braille using a mapping 
- Have a nice UI & animation when the Braille output is blank or generating
- Display the Braille output clearly in a styled result box
- Display all previously converted results in a grid layout
- When hovering over each result, show icon buttons for copy, download, and like


# Relevant docs
- Each cell consist of 3 rows and 2 columns. it has six fields in it
